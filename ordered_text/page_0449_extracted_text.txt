dekrementiranja se obično preklapaju na sledeći način:

  … 
  return *this; 
}

  operator++(); 
  return temp;

❖ Unarni logički operator negacije ! se preklapa za klase čiji 
se objekti koriste u kontekstu gde se očekuju Bulovi izrazi
❖ Zbog toga ove klase najčešće imaju i operator konverzije u 
tip bool koji je explicit, a operator ! svodi se na negaciju te 
konvertovane vrednosti:
X::operator bool () { 
  … 
} 
inline bool operator! (const X& x) { 
  return !bool(x); 
}
