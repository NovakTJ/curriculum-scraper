Konstruktori i destruktori
❖ Konstruktor (constructor) je posebna funkcija članica klase sa istim imenom kao i klasa, 
koja obezbeđuje inicijalizaciju objekata te klase:
class Clock { 
public: 
  Clock (int hh, int mm, int ss); 
  void tick (); // Tick a second 
  string getTime ();  // Returns current time 
  void setTime (int hh, int mm, int ss); 
private: 
  int h, m, s;  // Hour, minute, second 
}; 
Clock::Clock (int hh, int mm, int ss) { 
  setTime(hh,mm,ss); 
}

Konstruktori i destruktori
❖ Prilikom nastanka svakog objekta te klase (apsolutno bez izuzetka) poziva 
se konstruktor klase (jedan od više mogućih)
❖ Ovaj poziv obezbeđuje prevodilac implicitno (automatski):
Clock* pClk1 = new Clock(23,25,36); 
❖ Ako konstruktor klase zahteva argumente, onda se ovo ne može zaboraviti 
i izostaviti - prevodilac će prijaviti grešku:
Clock* pClk1 = new Clock; 
❖ Moguće je deﬁnisati i funkciju koja se uvek implicitno poziva kada 
objekat prestaje da živi (uništava se). 
Ova funkcija naziva se destruktor (destructor):
delete pClk1;
